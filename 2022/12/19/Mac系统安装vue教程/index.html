Mac系统安装vue教程

准备工作：安装node，npm

Vue依托nodejs，在安装Vue之前，先保证有装node,npm。通过如下命令检查是否有安装

node -v

npm -v

若没有安装，可选择手动安装or自动安装

手动安装需要自己去官网下载安装包，下载之后，直接点击运行，一路默认就行，直到安装完成。

node下载地址：https://nodejs.org/en/download/ （选择mac os）

自动安装终端执行brew install nodejs

安装Vue

  • 终端输入安装vue

npm i -g vue

  • 安装vue-cli

npm install -g vue-cli

  • 安装webpack

    (可检查自己电脑有没，我是安装的时候发现已经有webpack，通过执行 vue list 查看可以用的模板，如果有webpack，可自行跳过)

npm install -g webpack

  • 检查vue是否安装

vue -V

创建vue项目

vue init webpack myvue 该命令会在执行命令目录处创建myvue的文件夹

然后输入命令cd myvue

输入命令sudo npm install

发现如下爆错

网上search了一些解决方案，

  • 尝试将镜像设置成淘宝的，在执行发现同样的报错

npm config set registry http://registry.npm.taobao.org/

  • 尝试删除node_modules ，再执行 npm install也还是不行

  • 尝试安装cnpm，npm install -g cnpm --registry=https://registry.npm.taobao.org

​ cnpm install也还是报错

  • 最后搜到是mac m1的问题

    首先将node_modules文件夹删掉

    sudo cnpm i chromedriver --chromedriver_cdnurl=http://cdn.npm.taobao.org/dist/chromedriver

​

​ 最后sudo cnpm i

​

​ 运行vue，npm run dev

安装axios组件

npm install axios

安装axios组件报以下错误

npm ERR! Cannot read properties of null (reading ‘isDescendantOf’)

改用cnpm下载成功

sudo cnpm install axios
